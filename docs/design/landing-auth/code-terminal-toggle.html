> design notes

  • 메타포: VSCode/터미널의 settings prompt. 다른 4 프로젝트 메타포 (햇님↔달 / 양초 / 책 flip / 미니멀)와 격을 맞춘 Tech-Dark 전용.
  • 폭 jitter 0: 값은 항상 9 char ([ light ] vs [ dark ]). 토글해도 옆 요소 안 밀린다.
  • WCAG AA: light는 zinc-700/흰 7:1, dark는 zinc-100/ink-950 15:1. cyan 액센트는 라이트 cyan-700, 다크 cyan-neon (#22d3ee) 4.5:1↑.
  • caret: 단 하나만 (헤더의 hero caret 과 충돌 안 함 — 토글 안쪽 한정). reduced-motion 시 정적.
  • 키보드: role=switch + button 기본 동작 (Space/Enter). focus-visible 시 cyan 외곽선 + offset-1.
  • 호환: @getit/theme store (useTheme) 그대로 사용 — 다른 4 앱과 상태 sync.
